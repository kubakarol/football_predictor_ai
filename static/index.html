Football AI Predictor

Forma + siła rywali + miejsce meczu + kursy + prosty model ML

1. Wgraj dane CSV

Obsługuje Kaggle international results albo football-data.co.uk.

Najpierw wgraj CSV, potem trenuj.

2. Przewidź mecz

Tu pojawi się analiza.

3. Kursy z internetu

W .env ustaw ODDS_API_KEY. Sport key dla mundialu może się różnić, więc lista /odds/sports pomaga go znaleźć.

Bez klucza API zobaczysz komunikat.

4. Dopisz wynik po meczu

Po meczu dodaj wynik, a aplikacja odświeży stan formy/Elo. Model możesz przetrenować co np. 25 nowych meczów.

Tu zapiszesz wynik po meczu.
